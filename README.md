# Contafix
This script solves the issue of the phone numbers in Libya (AppleOSs)
Made by @BYTE-LY

This works only for Libyan phone numbers

how to use:
download this repo.
```
git clone https://github.com/BYTE-LY/Contafix
```
execute

```
cd Contafix
pip install -r requirements.txt
python contafix.py
```
